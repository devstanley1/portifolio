Pagina de Vendas

Sobre o Projeto

Página de vendas desenvolvida para um curso de programação front-end. O projeto apresenta um layout moderno, responsivo e focado em conversão, destacando os benefícios do curso e facilitando a inscrição do usuário. Utiliza HTML, CSS e JavaScript para criar uma experiência interativa e visualmente atraente, além de boas práticas de UI/UX para engajar futuros alunos.

Detalhes do Projeto

  • Tipo: Landing Page de Vendas
  • Objetivo: Conversão de usuários para inscrição em curso
  • Funcionalidades:
    • Layout responsivo e moderno
    • Formulário de inscrição interativo
    • Seções de benefícios e depoimentos
    • Animações suaves e elementos visuais atraentes
    • Otimização para conversão

Tecnologias Utilizadas

HTML5 CSS3 JavaScript
