Wave Music: Player de Música Moderno

Sobre o Projeto

Wave Music é um player de música web desenvolvido com foco em experiência do usuário e design moderno. O projeto oferece uma interface limpa e intuitiva para reprodução de músicas, com controles completos de playback, gerenciamento de playlist e visualização em tempo real das informações da faixa atual.

Detalhes do Projeto

  • Tipo: Aplicação Web de Entretenimento
  • Objetivo: Criar um player de música web moderno, responsivo e fácil de usar
  • Funcionalidades:
    • Reprodução, pausa e navegação entre faixas
    • Controle de volume com interface visual
    • Barra de progresso interativa com seek
    • Exibição de informações da música (título, artista, capa)
    • Design responsivo para diferentes dispositivos
    • Interface moderna com animações suaves

Tecnologias Utilizadas

HTML5 CSS3 JavaScript Web Audio API
